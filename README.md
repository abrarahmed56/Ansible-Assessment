# Ansible-Assessment
Take-home interview from Ansible, create a Python script running in Google Colab that converts the provided markdown meeting notes into a well-formatted Google Doc

[Assessment link](https://docs.google.com/document/d/15yLuwAJomRFE9av9JC3gDn0WN6xbZ_gkkNSJ531p8nE/edit?tab=t.0)

[Colab link](https://colab.research.google.com/drive/18YNew37LI_muuUqL1Gp5Hnj15NH476e3)

Open the Colab link and click "Run All"  
The first cell will open a window to authenticate with Google and allow permissions to the notebook to be able to create and edit a Google Doc  
The second cell will print a link to the newly created Google Doc. Click on this link to see the final result  
If all goes well, the last cell will output "Batch update successful!"
